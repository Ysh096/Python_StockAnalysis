# README

이 폴더는 '파이썬 증권 데이터 분석(김황후 저, 한빛미디어)' 책을 공부하며 정리하기 위해 만들었다. 다루게 될 내용은 다음과 같다.

- 파이썬 입문 <- 필요한 부분만 다루기
- 웹 스크래이핑
- 데이터 분석 툴(pandas, Scipy 등)
- 트레이딩 전략
- 자동 매매
- 딥러닝을 이용한 주가 예측



기본적으로 클론 코딩이지만, 중간중간 공부하며 알게 된 것들을 주석처럼 추가하여 나중에 읽기에도 용이하도록 작성 할 것이다. 또한, 가능하다면 소액으로 직접 자동 매매를 실시하여 그 성과를 측정해보는 것도 재미있을 것 같다.
